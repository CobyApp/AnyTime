# AnyTime

AnyTime은 시간 관리와 작업 관리를 위한 Flutter 앱입니다.

## 기능

- 시간 추적
- 작업 관리
- 안드로이드와 iOS 지원

## 개발 환경 설정

1. Flutter SDK 설치
2. 의존성 설치:
   ```bash
   flutter pub get
   ```

## 실행

```bash
flutter run
```

## 빌드

### 안드로이드
```bash
flutter build apk
```

### iOS
```bash
flutter build ios
```
